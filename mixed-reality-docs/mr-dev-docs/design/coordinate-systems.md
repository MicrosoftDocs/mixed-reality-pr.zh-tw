## <a name="spatial-coordinate-systems"></a><span data-ttu-id="e4055-191">空間座標系統</span><span class="sxs-lookup"><span data-stu-id="e4055-191">Spatial coordinate systems</span></span>

<span data-ttu-id="e4055-192">所有3D 圖形應用程式都會使用 [笛卡兒座標系統](/windows/uwp/graphics-concepts/coordinate-systems) 來表示虛擬物件的位置和方向。</span><span class="sxs-lookup"><span data-stu-id="e4055-192">All 3D graphics applications use [Cartesian coordinate systems](/windows/uwp/graphics-concepts/coordinate-systems) to reason about the positions and orientations of virtual objects.</span></span> <span data-ttu-id="e4055-193">這些座標系統會建立3個垂直軸，以放置物件： X、Y 和 Z 軸。</span><span class="sxs-lookup"><span data-stu-id="e4055-193">These coordinate systems establish 3 perpendicular axes along which to position objects: an X, Y, and Z axis.</span></span>

<span data-ttu-id="e4055-194">在 [混合的現實](../discover/mixed-reality.md)情況下，您的應用程式會有虛擬和實體座標系統的原因。</span><span class="sxs-lookup"><span data-stu-id="e4055-194">In [mixed reality](../discover/mixed-reality.md), your apps reason about virtual and physical coordinate systems.</span></span> <span data-ttu-id="e4055-195">Windows 呼叫的座標系統在實體世界中具有真正意義的 **空間座標系統**。</span><span class="sxs-lookup"><span data-stu-id="e4055-195">Windows calls a coordinate system that has real meaning in the physical world a **spatial coordinate system**.</span></span>
